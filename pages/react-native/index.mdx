## Why react native? 
Cross platform আপ্লিকেশন ডেভেলপমেন্ট করার জন্য react native ব্যবহার করা হয়ে থাকে। ReactJS এর উপরে ভিত্তি করে cross platform আপ্লিকেশন ডেভেলপমেন্ট করার জন্য react native ব্যবহার করা হয়। একই কোড ব্যবহার করে android & ios এর জন্য সহজেই মোবাইল অ্যাপ্লিকেশান বানানো যায়।
