## Why Route Groups

আমরা app folder এর মধ্যে similar route গুলো একই folder এর মধ্যে রেখে ভালোভাবে organize করতে পারি। কিন্তু আমরা জানি যে app folder এর কোন folder create করলেই সেটি route path হিসেবে কাজ করে, তাই এই default behaviour prevent করার জন্যই route groups করা হয়ে থাকে। নিচের file structure এ দেখা যাচ্ছে dashboard folder এর মধ্যে settings folder এবং analytics folder রয়েছে। তাই settings route access করতে হলে full url hobe `domain/dashboard/settings`। settings route এর পূর্বে dashboard route যুক্ত হয়ে যাচ্ছে। কিন্তু যদি সরাসরি settings route করতে চাই, তখনই route groups help করবে।

## Usages of Route Groups

যে folder কে group করতে চাই সেটা শুধুমাত্র parenthesis দিয়ে মুড়িয়ে দিলেই হয়ে যাবে। নিচের উদাহরণে দেখা যাচ্ছে dashboard folder কে parenthesis দিয়ে wrap করে দেওয়া হয়েছে। এর ফলে এই dashboard আর url এ দেখা যাবে না। settings page access করা যাবে এভাবে `domain/settings`
