## useMemo hook এর ব্যবহারঃ

useMemo hook useCallback এর মতই function cache করে থাকে। কিন্তু, পার্থক্য হচ্ছে return এর মধ্যে। useCallback এর মধ্যে যে function দেওয়া হয় সেটি cache করে রাখে আর useMemo এর মধ্যে যে function দেওয়া হয় সেটার result টি একটি variable এর মত return করে থাকে।
