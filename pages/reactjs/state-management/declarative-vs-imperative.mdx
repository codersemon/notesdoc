## `Declarative` vs `Imperative`
React `Declarative` way তে কাজ করে। এখানে Declarative বলতে নির্দেশনামূলক বোঝায়। অর্থাৎ, React এ আমরা শুধুমাত্র বলে দিই কখন কিভাবে UI তে পরিবর্তন করতে হবে, আর React নিজে থেকেই সেই শর্ত অনুযায়ী UI আপডেট করা দেয়। 

অন্যদিকে, `Imperative` বলতে নির্দেশনামূলক বোঝায়। অর্থাৎ, কখন UI তে পরিবর্তন করতে আর কোথায় আপডেট করতে হবে আমাদেরকে বলে দিতে হবে। React এ যেভাবে UI আপডেট করার দায়িত্ব React নিজেই পালন করে, কিন্তু Imperative way তে কাজ করলে সবকিছু আমাদেরকেই করতে হয়।
